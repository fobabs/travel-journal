## Run Locally

Clone the project

```bash
  git clone https://github.com/fobabs/travel-journal.git
```

Go to the project directory

```bash
  cd travel-journal
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev OR yarn dev
```
